# Personal Portfolio Website *In Progress*
A responsive personal portfolio designed to be a landing page to showcase projects, technical skills, and professional experience.

## **Tech Stack**
### **Frontend**
- HTML 
- CSS  
- JavaScript

### **Design & UI**
- Custom animations  
- Responsive layout  

### **Deployment**
- Ionos Web Hosting

## **Features**
### **Interface**
- Fully responsive design 
- Clean, minimal layout  
- Smooth UI transitions  

### **Visual Effects**
- Custom animated canvas background (“wiggle” effect)
- Subtle motion interactions
- Clear visual hierarchy  

### **Content**
- Project showcase  
- Technical skills display  
- Contact information  
- Professional branding  

## **Objective**
- Present a polished professional identity  
- Highlight engineering & development projects  
- Demonstrate frontend/UI capabilities  
- Provide a central hub for recruiters & collaborators

## **Design Principles**
- Minimal & modern aesthetic  
- Motion used intentionally, not distractingly  
- Readability & hierarchy first  
- Performance-conscious animations  


## **Future Enhancements**
- Dark / light theme toggle  
- Interactive project filtering
